# AI Goodreads Librarian 📖

A web application that transforms your Goodreads export into an interactive, conversational AI assistant. Chat with your personal reading history to get recommendations, recall your past reviews, and explore your reading trends.

A **sample Goodreads CSV export** is included in this repository for you to test the app quickly. Simply download this sample file and upload it directly to the live application: [AI Goodreads Librarian](https://demorag-mmm4tytjpmh9cmrjf7rzrf.streamlit.app/).
